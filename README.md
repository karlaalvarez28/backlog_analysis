# IT Support Backlog Analyzer

## Overview
As IT Service Desks scale, manually tracking ticket backlogs through Excel exports becomes highly inefficient. This Python project is an automation script designed to process raw ticket data (CSV exports), extract key operational metrics, and generate visual reports in seconds. 

It provides an immediate, clear picture of the current workload, allowing IT leaders to identify and take action on high-priority incidents faster.

## Business Value
When dealing with massive backlogs of thousands of assigned tickets, visibility is critical. This tool automates the reporting process, ensuring:
* **Time Efficiency:** Replaces manual filtering and counting with an instant script.
* **Proactive Management:** Highlights how many tickets are open, in progress, or resolved.
* **Data Visualization:** Uses a graphical representation obvious to stakeholders.

## Technologies Used
* **Python 3.x:** Core programming language.
* **`csv` module:** Native Python library used to parse and read the data export.
* **`matplotlib`:** Used to generate the professional bar chart for data visualization.

## How to Run It
1. Clone this repository to your local machine.
2. Ensure you have Python 64-bit installed.
3. Install the required library by running: `pip install matplotlib`
4. Run the script using the terminal:
   python backlog_analysis.py
